Documentation on the bot's API

Introduction

This is a simple to use, json api for the UnNamed bot and can be used for fetching simple server customizables, such as prefixes, user balances and work replies.

Base URL 

   <%- address %>/api

Balance Fetching

You can fetch user balances using this API. Just use the following url structure:

   /balance/[guild_id]/[member_id]
The API should return JSON with information about the user and their balances

Prefix Fetching

You can fetch guild prefixes using this API. Just use the following url structure:

   /prefix/[guild_id]
This will return simple JSON with the servers prefix

Work reply fetching

You can fetch guild work replies using this API. Just use the following url structure:

   /workreplies/[guild_id]
This will return an array filled with json containing reply IDs and responses

Currency fetching

You can fetch server currencies using this API. Just use the following url structure:

   /currency/[guild_id]
This will return json about the guilds currency
