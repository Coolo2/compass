Basic information

Compass is a discord bot made for customizability, advanced and fun commands, web integration and more. Most features will be listed here with descriptions on how to use them. Keep in mind that this bot is in beta, and gets constant updates and improvements

General questions

How do I add/remove work replies?

Beware: These functions require manage_server permissions.

Through web dashboard: Navigate to the web dashboard, and click on the server you want to edit. The tab you are set on by default is for adding and removing work replies. Near the server name you will notice buttons for addnig work replies. Any work replies listed below will have a delete button for deleting it.

Through the bot/comamnds: You can use the two below commands for adding and removing work replies. 

      <%- prefix %>addreply [response]
      <%- prefix %>removereply [replyID/all]

What are share IDs?

Share IDs are unique server specific identifiers for sharing your work reply setups. Sharing this will allow other servers to copy your work reply setup and sync it whenever they like. To get a Share ID navigate to your server in the web dashboard and if you have sufficient permissions it will be listed with the server name above the work replies. To get a server ShareID through commands, use: 

      <%- prefix %>sharereplies

How do I load another servers work replies?

Beware: These functions require manage_server permissions.

You can load a servers work replies with ShareIDs, more information about them can be found above. There are ways to load work replies through the web dashboard and through commands, here are how they are done:

Through the web dashboard: Navigate to the web dashboard, and click on the server you want to edit. You will be directed to the work replies page. Around the server name is a button saying "Load work replies from ShareID", type in the box to the left of this the ShareID of the server you want to copy on and click the button. You should have synced work replies!

Through the bot/comamnds: You can use the below command to load work replies from a ShareID 

      <%- prefix %>loadreplies [ShareID]

How do I disable the bot from working in a channel?

Beware: These functions require manage_channels permissions.

You can disable and enable the bot from working in certain channels through the web dashboard and through commands:

Through the web dashboard: Navigate to the web dashboard, and click on the server you want to edit. Near the top will be a list of different sections, click "channels". Through this page you can disable and enable channels!

Through the bot/comamnds: You can use the below command to enable or disable channels 

      <%- prefix %>disable [channel]
      <%- prefix %>enable [channel]
