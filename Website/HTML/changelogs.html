0.1.0 - Initial release (OCT-DEC)

This is the initial release of the bot and adds many features:

  • Bot with economy, moderation, fun and simple utility commands
  • Customizability
  • Extensive web dashboard covering most aspects of the bot which require a save.
  • Cross saving work replies and unique customizability features
  • Status pages and easy information about the bot through the website and discord server
